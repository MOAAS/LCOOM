Processes an interrupt, by checks the interrupt type. If it has received something, and there were no communication errors, adds all the bytes to a queue, by order of arrival. If it is ready to send something, clears the toSend queue (16 byte limit). If there was an error, discards the contents of the RBR (if there's any).
