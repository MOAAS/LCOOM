enum MouseEventType

Enumerates all the possible mouse events.

Enumerator
LB_PRESS 

Left button was pressed, and it previously wasn't.

RB_PRESS 

Right button was pressed, and it previously wasn't.

MB_PRESS 

Middle button was pressed, and it previously wasn't.

LB_RELEASE 

Left button was released, and it previously wasn't.

RB_RELEASE 

Right button was released, and it previously wasn't.

MB_RELEASE 

Middle button was released, and it previously wasn't.

MOUSE_MOVE 

No buttons were pressed, and the mouse reported movement.

BUTTON_EVT 

Any other button event.

Event_t GetEvent ( )

Waits until a notification is received, processes it to an Event_t struct. For the UART:

  • processes all the received bytes and assembles them into messages.
  • Will only generate event if the interrupt type is "receiver data"
