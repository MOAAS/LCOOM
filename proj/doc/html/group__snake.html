Detailed Description

Functions used to create and update a rainbow snake game.

◆ PowerupType

enum PowerupType

Enumerates the different types of snake Powerups.

Enumerator
ExtraHp 

Increases HP by 40% of Max HP (maximum of 100%)

Speedy 

Multiplies speed by 1.6.

Slowed 

Multiplies speed by 0.8.

Invincibility 

Grands 2 seconds of invincibility.

Rotate 

Turns the snake around.

Switcheroo 

Switchs the controls for 3 seconds.

Blind 

Does not allow the user to see the HP bar for 2 seconds.

BonusPoints 

Grants 5 times as many points for 3 seconds.

Bleeding 

Player loses health twice as fast for 1.5 seconds.

Random 

Grants a random powerup between the previous 9.
